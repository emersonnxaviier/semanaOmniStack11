# Como Usar

Para executar essa aplicação, é necessário o Git , Node.js + Yarn instalado em sua maquina.

# Instalação da API

```# Go into the repository
$ cd SemanaOmnistack11/backend

# Instalar dependências
$ yarn install

# Executar Migrates
$ yarn knex migrate:latest 

# Executar Seeds
$ yarn seed

# Executar à API
$ yarn dev```
